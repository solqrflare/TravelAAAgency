# TravelAAAgency — Satirical Tourism Ad

A satirical tourism advertisement for Antigua, created as a literary response to Jamaica Kincaid's *A Small Place*.

Built for an English literature assignment exploring irony, colonial legacy, and the tourist gaze.

## Live Site

Hosted via GitHub Pages: `https://solqrflare.github.io/TravelAAAgency/`

## Techniques Used

**Advertising Techniques:** Glittering Generality, Euphemism, Plain Folks, Transfer, Card Stacking

**Rhetorical Appeals:** Pathos, Ethos, Logos

**Literary Connection:** Jamaica Kincaid, *A Small Place* (1988)
